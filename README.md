# Multiplayer Memory Card Game

A real-time, responsive multiplayer memory card game with Apple-style aesthetics, animations, and real-time syncing via WebSockets.

## Project Structure

```
├── index.html
├── style.css
├── game.js
├── server.js
├── package.json
└── README.md
```

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start the Server:**
   ```bash
   npm start
   ```

3. **Play the Game:**
   Open `index.html` directly in two separate browser tabs or windows to start playing!
